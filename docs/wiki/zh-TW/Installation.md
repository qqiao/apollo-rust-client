## 安裝

### Rust

將以下內容新增到您的 `Cargo.toml` 檔案中：

```toml
[dependencies]
apollo-rust-client = "0.4.1" # 請確認並使用最新版本
```

或者，您可以使用 `cargo add`：

```bash
cargo add apollo-rust-client
```

### WebAssembly (npm)

```bash
npm install @qqiao/apollo-rust-client
```
